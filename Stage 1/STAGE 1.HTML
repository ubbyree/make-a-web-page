Lesson 1: The Basics of the Web and HTML

WORLD WIDE WEB

WWW is short for the World Wide Web. It is also commonly referred to as "the web". The web is a collection of interlinked documents contained on numerous computers across the world. A specific collection of documents on the web are referred to as web pages and are accessed by a "web browser" via the internet. When a web address is typed in a web browser a request is made by that computer to the server at the web address via HTTP. The server then locates the requested HTML document and returns it to the requesting computer. The HTML document is viewed via the web browser that requested the document.

HTML

HTML stands for Hypertext Markup Language. Most of the web is HTML documents.

HTML documents are made up of:

Text Content (what you see)
Markup (what it looks like)
References to other documents (ex: images & videos)
Links to other pages

MARKUP / TAGS & ELEMENTS

Markup is a code for formatting the layout and style of text by the application of numerous tags. Tags create elements and a collection of elements create a HTML document. Most tags have an opening, contents, and a closing. If you forget to close a tag then it will be applied to all text past it.

Some common tags are:
"b" which makes contents bold
"em" which makes contents italics
"a" (anchor) for making links

The exception to this rule are void tags. Void tags do not contain contents so they don't require closing.

Some common void tag are:
"br" which ends a line of text.
"img" defines an image

Some elements are inline and some are block. Block elements create a box that can have a width and a height.

ATTRIBUTES
Attributes specify a value for the contents of a tag. Some common attributes are:
"href" which is used with the anchor tag to specify the location of the created link. The contents of the tag will be the name of the link which will be viewable to the user.
"src" is used with the img tag to specify the url of the image to be used.

Lesson 2: Creating a Structured Document with HTML

IT'S ALL BOXES
HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. The invisible boxes have a width and a height. Most everything in an HTML document is a box.
INDENTATION OF HTML
HTML consists of a lot of nested boxes. The more indented the element the more nested the box is. Boxes are defined by DIV tags.
TEXT EDITORS FOR CODING
Programmers use special text editors to write code. An example would be Sublime or code pen. These editors can add ending tags automatically or let you see a live update of the web page while you change or modify code.

Lesson 3: Adding CSS Style to HTML Structure

CSS
CSS Stands for Cascading Style Sheets. CSS provides a way to control the look and style of HTML elements. For example background colors, fonts, text size are only a few ways to make the HTML code more pleasing to look at and easier to read.
AVOID REPETITION
Avoiding Repetition is one of the best ways to cut down on work content and avoid errors. For example you can use DIV tags to divide your HTML code into similar boxes and assign like boxes the same Class name. Then use CSS code to set attributes for each Class name. This is much less labor intensive than adding similar CSS code throughout the HTML code for each DIV tag. The more HTML code you have the more typing this saves you. And if you do find an error in your CSS code you can change it in one place and it will update all of the HTML code for that class.
